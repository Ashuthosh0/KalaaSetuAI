# KalaaSetuAI

## Overview

India is a land of timeless traditions, where classical musicians, dancers, artists, and spiritual experts like yoga teachers and pundits spend decades mastering their craft. Yet, in a digital-first world, these virtuosos often remain unseen and undervalued while eager learners and seekers struggle to find authentic Indian talent. KalaaSetu and KalaaSetuAI bridge this gap, making it easy to discover, connect, and empower artists and cultural experts across India—helping sustain cherished art forms for generations to come.


## Features

- **Artist Discovery:** Effortlessly search and discover authentic Indian artists, teachers, and experts.
- **Direct Booking:** Book musicians, dancers, spiritual mentors, and more for weddings, spiritual events, workshops, and personal enrichment.
- **Verified Profiles:** Each talent’s expertise is validated, ensuring authenticity and quality.
- **Personalized Recommendations:** Get matched with the right artists or mentors tailored to your needs.
- **Seamless Communication:** In-platform messaging connects seekers to talent safely and directly.(not yet implemented)
- **AI Image Enhancement:** AI enhances the image uploaded by artisans

## Tech Stack

- **TypeScript**
- **MongoDB**
- **Reactjs**
- **Node/Express**


## Getting Started

1. **Clone the repositories**
   ```bash
   git clone https://github.com/Ashuthosh0/KalaaSetuAI.git
   ```

2. **Install dependencies**
   Navigate to each project directory and run:
   ```bash
   npm install
   ```

3. **Set up environment variables**
   - Copy `.env.example` to `.env` in each repo (if applicable).
   - Fill in required API keys and configuration.

4. **Run the Development Server**
   ```bash
   npm run dev
   ```

5. **Explore the Platform**
   - Visit `http://localhost:3000` (or the port specified) to get started.
